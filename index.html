Advanced: build a redirect map from search results

Migrating a site and need a list of its indexed URLs? This snippet reads the search results currently displayed in your browser and copies their paths as two-column CSV, ready to paste above.

  1. Search for site:yourdomain.com in Google. Add &num=100 to the address bar to load more results per page.
  2. Open DevTools (F12) and switch to the Console tab.
  3. Click the button below, then paste into the console and press Enter. Chrome blocks pasted code the first time — if prompted, type allow pasting, press Enter, then paste again.
  4. Paste the result into the text area above. Tick First row is a header, then download.

Reads one page at a time. Only the results currently on screen are captured, so run it on each page of results and paste them one after another. Every target path is set to / as a placeholder — edit the second column to the real destinations before using the file.
